Hexo图片处理

发表于 2020-09-26 更新于 2020-09-27 阅读次数：

本地引用

绝对路径

当博客只用到少量图片时，可以将图片统一放在source/images文件夹中，通过markdown语法访问。

![](/images/name.jpg)

图片既可以在首页看到，又可以在正文中看到。

相对路径

图片可以放在文章同一目录对应的文件夹中，通过配置_config.yml来生成。

post_asset_folder:true

当$ hexo new post_name生成新文章时，在source/_posts中会生成文章post_name.md和资源文件夹post_name。将资源放入文件夹后就可以用相对路径引用了

![](image.jpg)

这样图片只能在文章中显示，无法在首页中显示。

需要标签插件语法

{% asset)img image.jpg This is an image %}
